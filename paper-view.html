Programming Fundamentals — Final 2023

01

What is a variable?

Describe the concept in the context of memory allocation and data types. (5 Marks)

Answer

A variable is a named storage location in computer memory that holds a value. When declared, memory is allocated based on the data type (e.g., int = 4 bytes, char = 1 byte). Variables provide a way to label and access data during program execution. They have a type, name, value, and scope.

02

Explain loops

Compare and contrast 'while' and 'for' loops with examples of use cases. (10 Marks)

Answer

A for loop is used when the number of iterations is known beforehand. Syntax: for(init; condition; update). Example: printing numbers 1-10.

A while loop is used when iterations depend on a condition being met. Syntax: while(condition). Example: reading user input until 'quit' is entered.

Both achieve repetition but differ in initialization and use case clarity.

03

What is an array?

Define arrays and explain how they are stored in memory. Provide a code example. (8 Marks)

Answer

An array is a collection of elements of the same data type stored in contiguous memory locations. Each element is accessed via an index starting from 0.

Example: int arr[5] = {10, 20, 30, 40, 50};
Here, arr[0] = 10, arr[4] = 50. Arrays provide O(1) access time and are fundamental for data manipulation.
